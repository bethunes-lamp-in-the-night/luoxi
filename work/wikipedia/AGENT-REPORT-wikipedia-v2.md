```wiki
| death_cause        = Complications following heart surgery
| known_for          = Medical incident leading to hospital department closure
}}
```

**Citation templates:**
```wiki
<ref>{{cite news|title=...|url=...|publisher=...|date=...|language=zh|access-date=...}}</ref>
```

**All 11 citations use proper parameters:**
- title (original Chinese or English)
- url (verified functional)
- publisher (with English translation in parentheses for Chinese sources)
- date (YYYY-MM-DD format)
- language (zh for Chinese, en for English)
- access-date (2026-01-14)

**Other markup:**
- `{{zh|c=徐洛熙}}` for Chinese character formatting
- `{{Reflist}}` for automatic reference list generation
- `== Section headers ==` (not markdown #)
- `[[Wikilinks]]` to related articles (proper capitalization, piped links where needed)
- `<ref></ref>` inline citations
